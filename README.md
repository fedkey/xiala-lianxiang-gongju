##安装说明

> 在Windows系统中安装

1. 下载安装包,windows版本对应项目文件夹: windows/

2. 解压软件包，并进行安装,和平时安装软件没有任何不同

3. 软件安装过程中可以选择是否创建桌面图标或快速启动栏，请自行选择
